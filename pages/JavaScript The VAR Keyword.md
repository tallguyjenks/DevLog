#### [[JavaScript The VAR Keyword]]

---

The `VAR` Keyword is a variable assignment declaration that is available globally in scope no matter where it is declared.

```js
var x = 10;
// Here x is 10
{
  let x = 2;
  // Here x is 2
}
// Here x is 10
```

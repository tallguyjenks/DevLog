

#### Spacing

| **LaTeX**    | _Code_         |
| :----------- | :------------- |
| $x\,y$       | `$x\,y$`       |
| $x\:y$       | `$x\:y$`       |
| $x\;y$       | `$x\;y$`       |
| $x \quad y$  | `$x \quad y$`  |
| $x \qquad y$ | `$x \qquad y$` |
| $x\!y$       | `$x\!y$`       |

Sometimes you may want some extra space between elements and a space wont work and you need something like `\,`: 50285a

`\Sigma \Delta` == $\Sigma \Delta$

v.s.

`\Sigma\,\Delta` == $\Sigma\,\Delta$

There is now a slight space between the two

- [[s.q.tsql]] to view all kinds of schema information on DBO's in a database use this quick query

```sql
USE FMDShop
SELECT * 
FROM INFORMATION_SCHEMA.TABLES
WHERE table_name LIKE '%StartingList%'
ORDER BY table_name
```

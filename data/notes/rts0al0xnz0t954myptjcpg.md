

A Cartesian Product that matches every unique value from one table to every unique value on another table for the complete list of every unique pairing:

| Table | One |
| ----- | --- |
| 1     | A   |
| 2     | B   |


| Table | Two |
| ----- | --- |
| 1     | C   |
| 2     | D   |


| Result | Table |
| ------ | ----- |
| 1      | AC    |
| 2      | AD    |
| 3      | BC    |
| 4      | BD    |

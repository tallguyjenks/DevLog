# [[LaTeX Figures]]

---

```latex
\\begin{wrapfigure}{\<orientation `r` or `l`\>}{\<space to take up like 3in\>}
	\<your figure here\>
\\end{wrapfigure}
```

to wrap a figure with text use the following code:

```latex
\begin{wrapfigure}
    <your figure here>
\end{wrapfigure}
```
